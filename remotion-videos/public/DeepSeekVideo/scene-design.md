## Scene 1: 开场引入

**构图方案**：全屏背景 + 居中标题布局

**背景层（BG）**：
- 基础渐变：`linear-gradient(135deg, #0d1117 0%, #161b22 50%, #1c2333 100%)`
- 弥散光球：
  - 左上方：蓝色 `rgba(88,166,255,0.15)`，缓慢浮动
  - 右下方：紫色 `rgba(247,120,186,0.1)`，反向浮动
- 装饰元素：网格线底纹（`opacity: 0.03`）

**主体层（Subject）**：
- 标题区：
  - "DeepSeek V4/R2"（主标题，48px，`font-black`，渐变文字 `#58a6ff` → `#79c0ff`）
  - "为何双双难产？"（副标题，32px，白色）
  - 开场白文字（底部，24px）
- 入场动画：
  - 主标题：spring 从下方滑入（stiffness: 100, damping: 20）
  - 副标题：延迟 10 帧后 spring 滑入
  - 开场白：延迟 20 帧后淡入

**前景层（FG）**：
- 装饰粒子：4-6 个浮动光点
- 动效：上下浮动 + 透明度脉冲

**动效时间线**：
| 帧范围 | 动画内容 |
|--------|----------|
| 0-30 | 背景弥散光淡入 |
| 10-40 | 主标题 spring 滑入 |
| 20-50 | 副标题 spring 滑入 |
| 30-60 | 开场白淡入 |

## 通用动效规范

### Spring 配置
```typescript
const springConfig = {
  stiffness: 100,
  damping: 20,
  mass: 1.2,
  overshootClamping: false
};
```

### 错位弹入（Staggering）
- 列表项之间延迟：8-10 帧
- 使用 `Math.max(0, frame - delay * index)`

### 高亮闪烁
```typescript
const highlight = 0.7 + Math.sin(frame * 0.15) * 0.3;
// 用于关键词或重要信息
```

### 字体大小规范
- 主标题：48px（8的倍数）
- 副标题：32px
- 正文：24px
- 注释：16px
