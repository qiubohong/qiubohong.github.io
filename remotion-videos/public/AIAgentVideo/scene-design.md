# AI Agent 视频场景设计方案

## 整体视觉风格

- **主题色**：科技蓝紫渐变（#58a6ff → #79c0ff）
- **强调色**：活力橙（#f0883e）
- **背景风格**：GitHub 暗色系 + 弥散光球
- **字体**：PingFang SC / Microsoft YaHei

---

## Scene 1: 开场 - AI Agent 核心转变

**构图方案**：居中构图，左右分屏对比

**背景层（BG）**：
- 渐变色：linear-gradient(135deg, #0d1117 0%, #161b22 50%, #1c2333 100%)
- 弥散光球：左侧红色系（#ff5555），右侧蓝色系（#58a6ff），缓慢脉动

**主体层（Subject）**：
- 左侧：传统 AI 图标（机器人 + 问号），带 ❌ 标记
- 右侧：AI Agent 图标（机器人 + 齿轮 + 工具），带 ✅ 标记
- 中间：大箭头 "→" 连接，渐变色动画
- 入场动画：左右卡片弹性滑入（spring: stiffness=100, damping=20）

**前景层（FG）**：
- 浮动粒子（蓝色光点），1.5x 速度飞过
- 关键词闪烁："答题机器" vs "会思考的执行者"

**动效时间线**：
| 帧范围 | 动画内容 |
|--------|---------|
| 0~30 | 背景弥散光淡入 |
| 10~40 | 左侧卡片滑入 |
| 20~50 | 右侧卡片滑入 |
| 35~60 | 中间箭头缩放入场 |
| 50~80 | 关键词高亮闪烁 |

**图片展示**：`/assets/img/ailearn/daily/17/1.png` - 全屏展示（可选）
