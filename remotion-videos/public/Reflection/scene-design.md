# Reflection 视频场景设计文档

## 整体风格

基于 PromptEngineering 系列统一设计规范：
- **背景色**：`linear-gradient(135deg, #0d1117 0%, #161b22 50%, #1c2333 100%)`（GitHub 暗色风格）
- **字体**："PingFang SC", "Microsoft YaHei", Arial, sans-serif（中文友好）
- **主标题渐变**：`linear-gradient(45deg, #58a6ff, #79c0ff)`（蓝色系）
- **强调色**：`#f0883e`（橙色）
- **辅助强调色**：`#ffd200`（金黄）
- **文字次要色**：`#c9d1d9`
- **文字弱化色**：`#8b949e`
- **卡片背景**：`rgba(255,255,255,0.06)`

## 场景列表

### Scene 1: Hook引入（黄金3秒）

**文案**：
> 之前我们学了 ReAct，让 AI 边想边做。今天学另一个神奇模式 —— Reflection！让 AI 学会"自我检讨"，做完任务回头看看哪里可以改进。

**构图方案**：居中布局，使用大字号对比制造视觉冲击

**背景层（BG）**:
- 使用 `scene1-bg.png`（已复制自 ReAct）
- 弥散光球：蓝色(#58a6ff)和紫色(#a371f7)调，缓慢移动

**主体层（Subject）**:
- 大标题 "Reflection"（渐变蓝色，64px）
- 副标题 "让 AI 学会自我反思"（24px，白色）
- 简短说明文字（20px，次要色）
- 核心关键词高亮："ReAct"、"自我检讨"

**动效时间线**：
| 帧范围 | 动画内容 |
|--------|---------|
| 0~30 | 背景淡入 |
| 10~50 | 标题 spring 滑入 |
| 40~70 | 副标题淡入 |
| 60~90 | 说明文字逐行出现 |

### Scene 2: 核心流程图

**文案**：
> Reflection 的核心流程就三步：生成、反思、优化。就像写代码：写完 run 一下，发现报错，回头改改再跑。看图，这是完整的工作循环。

**构图方案**：上方标题+下方大图展示

**背景层（BG）**:
- 使用 `scene2-bg.png`
- 弥散光球偏右上角

**主体层（Subject）**:
- 顶部标题 "Reflection 工作原理"（48px，蓝色）
- 主体：展示 `reflection-loop.png` 图片（占画面70%高度）
- 三个步骤标签卡片：生成（绿色#3fb950）、反思（橙色#f0883e）、优化（蓝色#58a6ff）

**布局要点**：
- 图片展示区域必须自适应，确保标签行可见
- 使用 `flexShrink: 0` 确保标签行不被压缩

**动效时间线**：
| 帧范围 | 动画内容 |
|--------|---------|
| 0~20 | 背景弥散光淡入 |
| 20~50 | 标题 spring 滑入 |
| 50~80 | 图片淡入+微缩放 |
| 80~120 | 三个步骤标签依次弹入（错位10帧）|

### Scene 3: 基础实现

**文案**：
> 最简单的实现就是两次调用模型。第一次生成答案，第二次让它反思，第三次根据反思优化。核心就是一个循环：生成→反思→优化。

**构图方案**：左侧文字说明 + 右侧代码展示（AI对话动画方案）

**背景层（BG）**:
- 使用 `scene3-bg.png`

**主体层（Subject）**:
- 左侧40%：分步骤说明（生成→反思→优化）
- 右侧60%：AI对话动画窗口（宽度从0动画展开到480px）
- 对话框展示简化的 Python 伪代码

**AI对话动画**：
- 用户消息（橙色右对齐）："请回答这个问题"
- AI消息（蓝色左对齐）：显示代码结构
- 打字机效果展示关键代码行

**动效时间线**：
| 帧范围 | 动画内容 |
|--------|---------|
| 0~30 | 左侧说明文字淡入 |
| 60~90 | 右侧对话框宽度展开（0→480px）|
| 90~150 | 对话气泡依次弹入+打字机效果 |

## 重要设计规则

### 字体大小规范
- 最小字号：16px
- 必须是8的倍数：16, 24, 32, 40, 48, 56, 64...
- 标题：48-64px
- 副标题：24-32px
- 正文：20-24px

### 布局安全规范
- 核心内容保持 `p-20`（80px）以上边距
- 避免文字贴近屏幕边缘
- 三分法构图，制造视觉张力

### 动效规范
- 入场使用 spring 动画
- 列表元素错位弹入（delay i*10帧）
- 禁止线性插值（特殊情况除外）
