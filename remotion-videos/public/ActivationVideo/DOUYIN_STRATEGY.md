## 💬 评论区运营

### 预设问题回复

**Q1: "ReLU 和 Leaky ReLU 有什么区别？"**

```
A: ReLU在负数区直接截断为0，可能导致神经元"死亡"。
Leaky ReLU在负数区保留微小梯度（通常0.01），
避免了这个问题。实际应用中Leaky ReLU更稳定！
```

**Q2: "Swish 真的比 ReLU 好吗？"**

```
A: 精度上Swish确实更高（95% vs 90%），
但计算效率ReLU更优。所以：
- 追求精度 → Swish
- 追求速度 → ReLU
- 移动端 → Swish（MobileNetV3首选）
```

**Q3: "初学者应该用哪个？"**

```
A: 强烈推荐ReLU！
原因：
1. 简单易懂
2. 90%的网络都在用
3. 效率最高
4. PyTorch/TensorFlow默认选择

等熟练后再尝试Swish等高级函数。
```
