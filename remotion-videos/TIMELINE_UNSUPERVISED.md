## 性能优化建议

1. **预渲染**: 使用 `npm run build:unsupervised` 预渲染视频
2. **缓存**: Remotion会自动缓存已渲染的帧
3. **并行渲染**: 使用 `--concurrency` 参数加速渲染

```bash
remotion render src/index.ts UnsupervisedLearningVideo out/unsupervised-learning.mp4 --concurrency=4
```
