# Chinese Writer De-AI

## 概述

这个技能用于将 AI 生成的中文文本转化为更自然、更有人情味的表达方式。

AI 写作常常有以下痕迹：

- 过度使用 "首先、其次、再者、最后、综上所述"
- 句子过长、结构过于规整
- 缺乏个人情感和口语化表达
- 使用大量套话和空洞词汇

本技能通过以下策略去除 AI 味：

1. **打破机械结构**：少用分点列举，多用自然过渡
2. **缩短句子长度**：将长句拆短，增加节奏感
3. **增加口语元素**：加入"其实"、"说实话"、"说实话"等口头禅
4. **注入个人视角**：加入第一人称视角和真实感受
5. **优化连接词**：用"而且"、"不过"、"问题是"替代"此外、然而"
6. **减少空洞修饰**：删除"非常、极其、显著"等过度修饰词

## 使用场景

- 小红书、公众号等自媒体文章润色
- 知乎回答、豆瓣影评等社区内容优化
- 个人博客、随笔等文字创作
- 任何需要"人味"的文字内容

## 处理流程

### 第一步：识别 AI 痕迹

检查文本中是否有以下特征：

- 机械化连接词：首先、其次、再者、最后、综上所述、总而言之
- 过度正式："本文将探讨"、"基于此"、"由此可见"
- 空洞修饰：大量的"非常"、"极其"、"显著地"
- 完美结构：段落过于对称，每段长度相近
- 缺乏人称：全文无"我"、"你"等代词

### 第二步：结构重组

- 打乱"总-分-总"的完美结构
- 允许段落长短不一
- 将列举改为叙述
- 适当增加冗余，模拟真实写作思维

### 第三步：语言优化

**连接词替换表**：
| AI 痕迹 | 自然替代 |
|---------|----------|
| 首先 | （删除或改为）要说这个 / 第一点 |
| 其次 | 还有就是 / 再说 |
| 此外 | 而且 / 另外 |
| 然而 | 不过 / 但是 |
| 因此 | 所以 / 结果就是 |
| 综上所述 | 说到底 / 总之 |
| 值得注意的是 | 其实 /
| 基于此 | 从这个角度来说 |

**句式优化**：

- 长句 → 短句（每句控制在 30 字以内）
- 被动句 → 主动句
- 书面语 → 口语化表达
- 增加问句和感叹句

### 第四步：注入人味

增加以下元素：

- 个人经历或观察
- 具体细节而非泛泛而谈
- 情绪波动（惊讶、困惑、兴奋）
- 适度的口语化停顿("嗯"、"啊"、"对吧")
- 不完美的表达，模拟真实写作

## 示例

### 示例 1：小红书风格

**原始文本（AI 痕迹明显）**：

```
本文将介绍三种提高学习效率的方法。首先，制定合理的学习计划至关重要。其次，保持良好的作息习惯有助于提升专注力。此外，适当休息能够避免过度疲劳。综上所述，通过以上方法可以有效提高学习效率。
```

**去 AI 味后**：

```
说实话，我自己试过好多所谓的"学习方法"，最后发现真正有用的其实就三点。

第一，计划别太满。我之前喜欢把每天排得满满当当，结果根本完不成，反而更焦虑。现在我会留一些空白时间，给突发情况和摸鱼（对，就是要摸鱼）。

第二，睡眠真的重要。熬过几次夜复习的人都知道，第二天脑子像浆糊一样，效率反而更低。我现在保证至少7小时，效果比熬夜好太多。

还有就是，别把自己逼太紧。学45分钟休息10分钟，比连续学3小时效率高得多。别问我怎么知道的，血泪教训。

总之，找到适合自己的节奏最重要。别人的方法再好，不适合你也白搭。
```

### 示例 2：知乎回答风格

**原始文本（AI 痕迹明显）**：

```
关于这个问题，可以从多个角度进行分析。首先，技术进步是推动行业发展的核心动力。其次，市场需求的变化也起到了关键作用。此外，政策环境的支持为行业发展提供了有力保障。基于以上分析，可以得出结论：该行业未来发展前景良好。
```

**去 AI 味后**：

```
这个问题我研究过一段时间，说说我的看法。

技术确实是第一推动力，这个毋庸置疑。但我觉得很多人忽略了一个点——其实是市场的需求倒逼了技术进步，而不是反过来。

举个例子，AI 翻译现在这么火，不是因为技术突然突破了（虽然确实有突破），而是因为跨境电商、跨国交流的需求爆发式增长。没有这个需求，技术再好也白搭。

政策这块嘛，支持力度确实大，但我觉得也别太乐观。政策是风向标，不是发动机，最终还是要看市场买不买账。

至于发展前景，我个人是看好的，但不是那种爆炸式增长，而是稳步上升。毕竟这个行业涉及面太广，不可能一蹴而就。
```

### 示例 3：公众号文章风格

**原始文本（AI 痕迹明显）**：

```
在当今社会，人们面临着前所未有的压力。有效管理压力对个人健康至关重要。本文将分享五个实用的减压技巧，帮助读者更好地应对生活压力。
```

**去 AI 味后**：

```
现代人压力有多大？这么说吧，我身边十个朋友，九个在失眠，还有一个在假装不失眠。

对抗压力这件事，我试过冥想、瑜伽、心理咨询，最后发现最有效的办法其实特别简单——就是学会"摆烂"。

不是真的躺平不干了，而是允许自己偶尔搞砸，允许事情不完美。以前我特别追求完美，事事要做到120分，结果把自己搞得精疲力竭。

现在我的原则是：重要的事做到80分，其他事及格就行。你猜怎么着？不仅压力小了，效率反而更高了。

分享几个我亲测有效的减压小技巧...
```

## 注意事项

1. **保留原意**：去 AI 味不等于乱改，核心观点必须保留
2. **适度即可**：不要过度口语化，保持适当的文字质感
3. **符合场景**：不同平台（小红书/知乎/公众号）调整语气和风格
4. **避免过度**：不要矫枉过正，避免从"AI 味"变成"做作味"
