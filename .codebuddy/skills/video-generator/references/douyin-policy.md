# 抖音社区自律公约 - 视频文案合规规则

> 来源：《抖音社区自律公约》（2022 年 09 月 13 日生效）
> 本文件提炼与视频文案创作最相关的合规要求，供生成文案时进行自检。

## 一、文案用词禁区

### 1. 禁止低俗媚俗用语（第 17 条）

以下词汇/表达**严禁**出现在标题、文案、字幕、标签中：

| 违规词汇             | 合规替换                     |
| -------------------- | ---------------------------- |
| 白嫖                 | 免费获取 / 零成本 / 免费领取 |
| 薅羊毛               | 领取福利 / 获取优惠          |
| 骚操作               | 实用技巧 / 高效方法          |
| 装 X                 | 展示技能 / 提升能力          |
| 傻 X / 蠢货 等骂人词 | 禁止使用                     |
| 性暗示词汇           | 禁止使用                     |

**判断标准**：使用粗俗语言、随意侮辱谩骂、展示恶俗行为的内容均属违规。

### 2. 禁止夸大宣传（第 33 条）

对产品或服务的性能、功能、质量做**虚假或引人误解**的商业宣传：

| 违规表达                  | 合规替换                   |
| ------------------------- | -------------------------- |
| 够用好几年                | 足够日常学习使用           |
| 永久免费                  | 当前免费（以官方政策为准） |
| 100% 有效 / 绝对管用      | 经过验证的方法 / 实测有效  |
| 秒变大神 / 立刻精通       | 快速入门 / 提升效率        |
| 治愈 / 根治（非医疗场景） | 改善 / 优化                |
| 保证赚钱 / 稳赚不赔       | 禁止使用                   |

**判断标准**：对商品安全性能进行没有合理依据的断言或保证，将科学上未定论的观点当作定论用于宣传，均属违规。

### 3. 禁止不实信息（第 21 条）

- 不得编造或传播未经证实的信息
- 不得夸大灾害事件严重程度
- 不得虚构恐怖事件或危害公众安全事件
- 引用数据、功能说明必须真实准确

### 4. 禁止诱导性内容（第 35 条）

- 不得承诺"快速赚钱"、"高额回报"
- 不得以虚假演绎骗取流量、关注
- 不得使用标题党、炒作负面话题

## 二、标签（Hashtag）规范

### 禁止使用的标签类型

- 含低俗词汇的标签（如 `#白嫖指南`、`#薅羊毛攻略`）
- 含夸大宣传的标签（如 `#永久免费`、`#稳赚不赔`）
- 含政治敏感词汇的标签
- 含仇恨歧视词汇的标签

### 推荐标签风格

```
✅ #AI教程  #免费教程  #免费工具  #AI学习  #技术分享
✅ #零成本  #入门指南  #实用技巧  #效率工具
❌ #白嫖  #薅羊毛  #永久免费  #稳赚
```

---

## 三、内容价值导向规范（第 23 条）

### 禁止传播的价值观

- 宣扬流量至上、奢靡享乐、炫富拜金
- 展示超额消费、过度消费，打造富豪人设
- 宣扬厌世情绪、丧文化及消极负面亚文化
- 以卖惨、炫富、引战、恶搞等方式炒作

### 鼓励的内容方向

- 传递真实、有价值的知识和技能
- 积极向上、帮助用户解决实际问题
- 内容真实，不夸大、不虚构

---

## 四、文案合规自检清单

在生成或审核文案时，逐项检查：

```
□ 标题中是否含有低俗词汇（白嫖、薅羊毛等）？
□ 文案中是否有夸大宣传（够用好几年、永久免费等）？
□ 数据和功能描述是否真实准确？
□ 标签是否含有违规词汇？
□ 是否有诱导性承诺（快速赚钱、稳赚不赔等）？
□ 内容是否传递积极正向的价值观？
□ 是否有性暗示、低俗联想的词汇或表达？
```

## 五、常见违规案例与修正示例

### 案例 1：标题违规

```
❌ 违规：龙虾 AI 不花一分钱！免费白嫖 9000 万 tokens！
✅ 合规：龙虾 AI 零成本启动！三平台免费额度超 9000 万 tokens！
```

### 案例 2：文案夸大宣传

```
❌ 违规：三个平台都注册，够用好几年！
✅ 合规：三个平台都注册，足够日常学习使用！
```

### 案例 3：标签违规

```
❌ 违规：#白嫖指南 #薅羊毛
✅ 合规：#免费教程 #AI学习
```

### 案例 4：功能夸大

```
❌ 违规：学会这招，你的 AI 效率提升 100 倍！
✅ 合规：学会这招，你的 AI 使用效率大幅提升！
```
