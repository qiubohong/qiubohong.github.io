学会下指令之后，你可能会发现一个问题：

> "我让龙虾帮我查一下飞书日历，它说'我没有访问飞书的能力'……"

这很正常。**刚装好的龙虾，就像一部刚买来的新手机——系统是有了，但 App 还没装。**

这篇文章就来解决这个问题：给龙虾装上"技能包"（Skills），让它从一个只会聊天的助手，变成真正能帮你干活的全能助手。

<!-- more -->

# 是什么

## Skills 是什么？

Skills（技能包）就是龙虾的"App"。

你的手机出厂时只有基础功能（打电话、发短信），但装上微信、支付宝、高德地图之后，能做的事情就多了几十倍。

![Skills概念对比图](/assets/img/ailearn/openclaw/05/skills_concept.png)

**一句话：Skills 决定了龙虾能做什么。**

## Skills 从哪里来？

Skills 有三个来源：

**1. 官方技能库（推荐新手用这个）**

- 龙虾官方提供的技能，质量有保障
- 一键安装，简单方便
- 有几十个常用技能，够用了

**2. 社区技能库（等熟练了再用）**

- 其他用户分享的技能，种类很多
- 但质量有好有坏，需要自己判断

**3. 自己写技能（技术要求高）**

- 完全按照自己的需求定制
- 需要会写代码，不适合新手

**新手建议**：只用官方技能库就够了，里面的技能已经能解决大部分日常需求。

## 必装的 5 个技能包

根据使用频率和实用程度，这 5 个技能包是最值得安装的：

| 技能包名称             | 能做什么                                     | 安装难度 |
| ---------------------- | -------------------------------------------- | -------- |
| **文件系统（fs）**     | 读写本地文件、整理文件夹、批量重命名         | ⭐       |
| **网页搜索（search）** | 搜索网页、抓取网页内容、查询实时信息         | ⭐       |
| **飞书（feishu）**     | 读写飞书文档、发消息、管理日历、操作多维表格 | ⭐⭐     |
| **浏览器（browser）**  | 自动打开网页、填写表单、截图、模拟点击操作   | ⭐⭐     |
| **代码执行（code）**   | 运行 Python/JS 代码，处理数据、生成图表      | ⭐⭐⭐   |

> 💡 **新手建议**：先装前两个（文件系统 + 网页搜索），这两个最简单，装完立刻能感受到龙虾能力的提升。飞书技能需要你有飞书账号，浏览器技能需要额外安装一个插件，代码执行技能需要电脑装了 Python。

# 怎么做

## 手把手：安装你的第一个技能包

我们以安装"网页搜索（search）"技能为例，一步步来。

![技能安装流程图](/assets/img/ailearn/openclaw/05/installation_flow.png)

**安装完成后**，技能状态会显示为 **"已启用"**。

### 第三步：测试技能是否生效

回到对话界面，发送这条消息：

```
帮我搜索一下"2026年春节档票房排行"，把前3名告诉我。
```

如果龙虾真的搜索了网页并给你返回了结果（而不是说"我不知道"），说明技能安装成功！🎉

---

## 安装"文件系统"技能

文件系统技能让龙虾能读写你电脑上的文件，是最基础也最常用的技能之一。

### 安装步骤

1. 在技能管理页面搜索 `fs` 或 `file system`
2. 找到 **"File System"**，点击 **"一键安装"**
3. 安装完成后，**需要设置文件访问权限**（保护隐私安全）

### 重要：设置文件访问权限

安装文件系统技能后，**必须告诉龙虾它能访问哪些文件夹**，否则它什么都做不了（这是为了保护你的隐私）。

![权限配置示意图](/assets/img/ailearn/openclaw/05/permission_config.png)

**安全提醒**：根据官方安全建议，只给龙虾开放你真正需要的文件夹，不要让它访问整个电脑或系统文件夹。高风险操作建议开启审批模式。

### 测试文件系统技能

发送这条消息：

```
帮我在桌面上新建一个文件夹，名字叫"龙虾测试"，
然后在里面新建一个文本文件，内容是"Hello，龙虾！"，文件名叫"hello.txt"。
```

如果桌面上真的出现了"龙虾测试"文件夹，里面有"hello.txt"，说明文件系统技能工作正常。

---

## 安装"飞书"技能

飞书技能是职场人最常用的技能之一，装完之后龙虾可以帮你读写飞书文档、发消息、管理日历。

### 安装步骤

1. 在技能管理页面搜索 `feishu` 或 `飞书`
2. 找到 **"Feishu"**，点击 **"一键安装"**
3. 安装完成后，**需要完成飞书授权**（关键步骤）

### 飞书授权（关键步骤）

飞书技能需要你授权龙虾访问你的飞书账号，就像你授权一个 App 访问你的微信一样。

![飞书授权流程图](/assets/img/ailearn/openclaw/05/feishu_auth.png)

**授权成功后**，技能状态会显示为 **"已启用"**。

### 测试飞书技能

发送这条消息：

```
帮我查一下我今天飞书日历上有哪些日程，用列表告诉我。
```

如果龙虾真的列出了你今天的飞书日程，说明飞书技能工作正常。

---

## 安装"浏览器"技能

浏览器技能让龙虾能自动操控浏览器——打开网页、点击按钮、填写表单、截图……这是最"神奇"的技能之一。

**安装步骤**：

**根据官方文档，浏览器技能安装相对复杂，需要额外配置：**

**第一步：安装技能本体**

在技能管理页面搜索 `browser`，找到 **"agent-browser"**（官方技能），点击 **"一键安装"**。

**第二步：配置浏览器插件（关键步骤）**

![浏览器插件安装图](/assets/img/ailearn/openclaw/05/browser_plugin.png)

> 💡 **重要**：确保插件显示"已连接"状态，否则浏览器技能无法正常工作。

> ⚠️ **注意**：浏览器技能配置相对复杂，新手建议先安装其他基础技能，熟练后再尝试。

### 测试浏览器技能

发送这条消息：

```
帮我打开百度，搜索"今日天气"，把搜索结果页面截图发给我。
```

如果龙虾自动打开了浏览器、搜索了天气、并把截图发给你，说明浏览器技能工作正常。

# 经典案例

## 装完技能包之后，能做哪些以前做不到的事？

### 案例一：自动整理下载文件夹（需要文件系统技能）

**场景**：下载文件夹里乱七八糟，各种文件堆在一起。

**指令**：

```
帮我整理"下载"文件夹，规则如下：
- 视频文件（.mp4/.avi/.mkv）→ 新建"视频"子文件夹
- 图片文件（.jpg/.png/.gif）→ 新建"图片"子文件夹
- 文档文件（.pdf/.doc/.docx）→ 新建"文档"子文件夹
- 安装包（.exe/.dmg/.pkg）→ 新建"安装包"子文件夹
- 其他文件 → 不动

整理完告诉我每类移动了多少个文件。
```

**效果**：龙虾自动扫描下载文件夹，按类型分好，整个过程你只需要等着。

---

### 案例二：每天自动生成工作日报（需要飞书技能）

**场景**：每天下班前要写日报，总是忘或者懒得写。

**指令**：

```
每天下午 5:50，帮我做以下事情：
1. 读取今天飞书"工作记录"文档里的内容
2. 根据内容生成一份工作日报，格式：
   - 今日完成事项（列表）
   - 遇到的问题（如果有）
   - 明日计划（列表）
3. 把日报写入飞书"日报记录"文档，标题格式：[日期]工作日报
4. 在飞书群"团队日报"里发一条消息，内容是日报的摘要（不超过100字）
```

**效果**：每天下班前，龙虾自动生成日报、写入文档、发到群里，你完全不用操心。

---

### 案例三：监控竞品网站更新（需要网页搜索 + 浏览器技能）

**场景**：想及时了解竞品的最新动态，但每天手动去看太麻烦。

**指令**：

```
每天早上 9:00，帮我检查以下网站是否有新内容：
- https://competitor1.com/blog
- https://competitor2.com/news

如果有新文章或公告，把标题、发布时间、链接整理成列表，发到我的飞书私信。
如果没有更新，不用通知我。
```

**效果**：龙虾每天自动巡逻竞品网站，有更新才通知你，省去了大量手动查看的时间。

---

### 案例四：一键生成周报（需要文件系统 + 飞书技能）

**场景**：每周五要写周报，需要整理一周的工作内容。

**指令**：

```
每周五下午 5:00，帮我生成本周工作周报：
1. 读取本周（周一到周五）飞书日历上的所有日程
2. 读取飞书"工作记录"文档里本周的内容
3. 综合以上信息，生成周报，包含：
   - 本周完成的主要工作（按项目分类）
   - 本周参加的重要会议（从日历提取）
   - 下周计划
4. 保存为 Word 文档，文件名"[日期范围]周报"，存到桌面
5. 同时把周报内容发到飞书"周报"文档
```

**效果**：每周五下班前，一份完整的周报自动生成好了，你只需要检查一遍就能提交。

### 第二件：安装"文件系统"技能，整理一次桌面（10 分钟）

1. 安装 "File System" 技能
2. 设置桌面文件夹的读写权限
3. 发消息：`帮我看看桌面上有哪些文件，按类型列出来`

### 第三件：规划你的"技能安装清单"

根据你的日常需求，想想你最需要哪些技能：

- 经常用飞书 → 安装飞书技能

**把你的清单写下来，按优先级一个个装。** 不要一次装太多，先装最需要的，用熟了再装下一个。

# 进阶知识

## 技能是怎么工作的？

很多人好奇：技能装上去之后，龙虾是怎么"学会"新能力的？

**简单来说**：技能就像是龙虾的"工具箱"，龙虾的大脑负责判断什么时候该用哪个工具。

![技能工作流程图](/assets/img/ailearn/openclaw/05/skill_workflow.png)

**打个比方**：没有技能的龙虾就像是一个很聪明但没带工具的人，知道怎么修东西但没工具；装了技能的龙虾就是带着工具箱的修理工，什么都能修。

## 技能冲突怎么办？

有时候安装多个技能后，龙虾不知道该用哪个技能来处理任务。

**解决方法**：

1. **在指令里明确指定**：`用"文件系统"技能帮我读取桌面上的"报告.pdf"`
2. **技能优先级规则**：工作区技能 > 共享技能 > 内置技能

**安全建议**：高风险技能（如 `exec`、`shell`）建议开启审批模式，防止恶意操作。

这样龙虾就知道该用哪个技能了。

## 技能的安全边界

每个技能都有自己的"权限范围"，龙虾只能在这个范围内操作：

![安全边界示意图](/assets/img/ailearn/openclaw/05/security_boundary.png)

**这些限制是保护你的**。如果某个技能要求超出正常范围的权限，要谨慎安装。

# 总结

给龙虾装技能包，只需要记住三件事：

1. **Skills 是龙虾的"App"**：没有技能，龙虾只能聊天；装了技能，龙虾才能真正帮你干活。新手先装"网页搜索"和"文件系统"这两个最基础的。

2. **安装很简单，配置要认真**：技能安装本身一键搞定，但安装后的权限配置很重要——只给龙虾它真正需要的权限，不要无限制开放。

3. **按需安装，用熟一个再装下一个**：不要一次装太多，先装最需要的，用熟了再扩展。
