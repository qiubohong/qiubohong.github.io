# 是什么

## 每日信息助手能做什么？

先来看看，配置好之后，你的龙虾每天能帮你做哪些事：

![每日信息助手功能全景图](/assets/img/ailearn/openclaw/07/daily_assistant_overview.png)

| 功能类型    | 具体能做什么                                 | 推送到哪里       |
| ----------- | -------------------------------------------- | ---------------- |
| 🌤️ 天气推送 | 今日天气、穿衣建议、出行提醒                 | QQ / 飞书 / 微信 |
| 📰 新闻摘要 | 科技、财经、行业热点，每条 2-3 句话总结      | QQ / 飞书 / 微信 |
| 📅 日程提醒 | 今日待办、会议提醒、重要截止日期             | QQ / 飞书 / 微信 |
| 🔍 查资料   | 搜索任何话题，给你一份简洁的摘要             | 对话回复         |
| 🌐 翻译     | 中英互译、专业术语翻译、整段文字翻译         | 对话回复         |
| 📖 解释词汇 | 解释专业词汇、缩写、行业术语，用大白话说清楚 | 对话回复         |

**这些功能的核心**：龙虾会在你设定的时间自动执行，或者你随时发指令触发。

## 为什么要用龙虾来做这件事？

你可能会问：天气 App 不是已经有推送了吗？新闻 App 也有推送……

**龙虾的不同之处**：

1. **全部整合到一个地方**：天气 + 新闻 + 日程，一条消息搞定，不用打开三个 App
2. **内容可以定制**：只推送你关心的行业新闻，不是算法推给你的内容
3. **可以加上你的个人情况**：比如"今天有重要会议，提醒我提前 30 分钟准备"
4. **随时可以追问**：看到不懂的词，直接问龙虾，它能结合上下文解释

## 需要哪些技能？

实现每日信息助手，需要安装以下技能（如果还没装，参考[第 5 篇：装上技能包](https://qborfy.com/ailearn/openclaw/05.html)）：

![所需技能清单](/assets/img/ailearn/openclaw/07/required_skills.png)

| 技能名称              | 用途                         | 必须/可选 |
| --------------------- | ---------------------------- | --------- |
| `fetch` / `web-fetch` | 获取天气、新闻等网络信息     | 必须      |
| `feishu`（飞书）      | 推送消息到飞书               | 二选一    |
| `qq-bot`（QQ 机器人） | 推送消息到 QQ                | 二选一    |
| `search`              | 搜索网页，查资料用           | 推荐      |
| `translate`           | 翻译功能                     | 推荐      |
| `scheduler`           | 定时任务，让龙虾自动定时执行 | 推荐      |

> 💡 **飞书 vs QQ，选哪个？**
>
> - 工作场景用飞书，消息更正式，适合推送工作相关信息
> - 个人日常用 QQ，更轻松，适合推送生活信息
> - 两个都装也没问题，可以分类推送

### 快速安装所需技能

如果还没有安装上述技能，直接把以下指令发给龙虾，它会帮你一次性安装好：

**推送到飞书（推荐）**：

```
龙虾，帮我安装 fetch、feishu、search、translate、scheduler 这几个 skill 技能
```

**推送到 QQ**：

```
龙虾，帮我安装 fetch、qq-bot、search、translate、scheduler 这几个 skill 技能
```

**两个都要**：

```
龙虾，帮我安装 fetch、feishu、qq-bot、search、translate、scheduler 这几个 skill 技能
```

> 💡 安装完成后，飞书和 QQ 技能还需要做一次授权配置，下面「第一步」会详细说明。

# 怎么做

## 第一步：配置推送渠道

### 方案一：推送到飞书

**前提**：已安装飞书技能，并完成授权（参考第 5 篇）

飞书推送有两种方式：

**方式 A：推送到飞书机器人（推荐，最简单）**

1. 在飞书里创建一个群（或者用已有的群）
2. 在群里添加「自定义机器人」：群设置 → 机器人 → 添加机器人 → 自定义机器人
3. 复制机器人的 Webhook 地址（格式类似 `https://open.feishu.cn/open-apis/bot/v2/hook/xxxxx`）
4. 在龙虾的飞书技能配置里，填入这个 Webhook 地址

**方式 B：推送到飞书私信**

直接通过飞书技能发私信给自己，不需要额外配置，但需要完成飞书账号授权。

---

### 方案二：推送到 QQ

**前提**：已安装 QQ 机器人技能

QQ 推送需要一个「QQ 机器人账号」来发消息：

1. 准备一个备用 QQ 号（不要用主号，避免被封）
2. 在龙虾的 QQ 机器人技能里，扫码登录这个备用 QQ 号
3. 让这个备用 QQ 号加你的主 QQ 为好友，或者加入你的 QQ 群
4. 配置推送目标：你的主 QQ 号 或 QQ 群号

> ⚠️ **注意**：QQ 机器人有被封号风险，建议用备用号，不要用主号。发消息频率不要太高（每天 1-3 条为宜）。

## 第二步：设置每日天气推送

### 天气推送指令模板

把以下指令保存起来，每天早上发给龙虾，或者配置成定时任务：

```
帮我查询今天[城市名]的天气，然后推送到[飞书/QQ]。

推送内容格式如下：
🌤️ 今日天气播报
📍 城市：[城市名]
🌡️ 温度：[最低温]~[最高温]℃
🌂 天气：[天气状况]
👔 穿衣建议：[根据温度给出建议]
⚠️ 特别提醒：[如有大风/降雨/高温等极端天气，给出提醒]

数据来源：中国天气网 (weather.com.cn)
```

**实际效果示例**：

```
🌤️ 今日天气播报
📍 城市：深圳
🌡️ 温度：22~28℃
🌂 天气：多云转晴
👔 穿衣建议：早晚凉，建议穿薄外套；中午可换短袖
⚠️ 特别提醒：无特殊天气，出行愉快！
```

## 第三步：设置每日新闻摘要

### 新闻摘要指令模板

```
帮我搜索今天的[科技/财经/行业]新闻，选出最重要的 5 条，
每条用 2-3 句话总结，然后推送到[飞书/QQ]。

推送内容格式如下：
📰 今日新闻摘要（[日期]）

1️⃣ [新闻标题]
   [2-3句摘要]
   🔗 来源：[媒体名称]

2️⃣ [新闻标题]
   [2-3句摘要]
   🔗 来源：[媒体名称]

（以此类推，共5条）

---
以上内容由龙虾助手整理，仅供参考。
```

**定制你的新闻偏好**：

在指令里加上你的具体需求，龙虾会按照你的偏好筛选：

```
我关注的领域：AI 技术、互联网行业、A 股市场
不感兴趣的内容：娱乐八卦、体育赛事
新闻来源偏好：36氪、虎嗅、财联社
```

## 第四步：设置日程提醒

### 日程提醒指令模板

日程提醒需要你先告诉龙虾你的日程，然后让它帮你整理并推送：

```
我今天的日程如下：
- 09:00 部门周会（会议室 A）
- 14:00 和客户视频通话（腾讯会议）
- 17:00 提交周报
- 截止日期：明天上午 10:00 前提交项目方案

帮我整理成今日日程提醒，推送到[飞书/QQ]。

推送格式：
📅 今日日程提醒（[日期] [星期]）

⏰ 09:00 部门周会
   📍 地点：会议室 A
   💡 建议：提前 10 分钟到场

⏰ 14:00 客户视频通话
   📍 平台：腾讯会议
   💡 建议：提前测试网络和摄像头

⏰ 17:00 提交周报
   💡 建议：16:30 开始整理

⚠️ 截止提醒：明天 10:00 前提交项目方案，今天需要完成初稿！
```

> 💡 **进阶用法**：如果你用飞书日历，可以让龙虾直接读取你的飞书日历，自动生成日程提醒，不需要手动输入。

## 第五步：设置定时自动推送

### 使用 scheduler 技能实现自动化

有了 `scheduler` 技能，你可以让龙虾在每天固定时间自动执行上面的推送任务，不需要你手动触发。

**配置方法**：

1. 打开龙虾管理界面（`http://localhost:18789`）
2. 进入「技能管理」→ 找到 `scheduler` 技能 → 点击「配置」
3. 添加定时任务，填写以下信息：

```
任务名称：每日早报推送
执行时间：每天 07:30
执行指令：
  帮我查询今天深圳的天气，搜索今天的科技新闻（选5条），
  整合成早报格式，推送到飞书机器人。

  早报格式：
  ☀️ 早上好！今日早报来了

  【天气】[天气信息]

  【今日新闻】
  [5条新闻摘要]

  祝你今天工作顺利！🎉
```

![定时任务配置示意图](/assets/img/ailearn/openclaw/07/scheduler_config.png)

**常用定时时间建议**：

| 任务              | 建议时间 | 原因                       |
| ----------------- | -------- | -------------------------- |
| 早报（天气+新闻） | 07:30    | 上班路上看，开始一天的节奏 |
| 日程提醒          | 08:00    | 到公司前看，心里有数       |
| 下午新闻          | 14:00    | 午休后，了解下午的动态     |
| 晚间总结          | 18:00    | 下班前，回顾今天的重要事项 |

## 第六步：随时查资料、翻译、解释词汇

这部分不需要额外配置，直接发指令给龙虾就行。

### 查资料

```
帮我搜索"大语言模型的 RAG 技术"，用大白话解释给我听，
不超过 300 字，最后给我 2-3 个延伸阅读的链接。
```

### 翻译

```
帮我翻译以下内容（中译英，保持专业语气）：
[粘贴要翻译的内容]
```

```
帮我翻译以下英文邮件，翻译成中文，
同时告诉我这封邮件的主要意思和对方的态度：
[粘贴英文邮件]
```

### 解释专业词汇

```
帮我解释以下词汇，用大白话，举一个生活中的例子：
- RAG（检索增强生成）
- Token（大模型里的 Token）
- Fine-tuning（微调）
```

**实际效果示例**：

```
📖 词汇解释

RAG（检索增强生成）
大白话：就像你考试时可以翻书一样。
普通 AI 只能靠记忆回答问题，RAG 让 AI 在回答前先去"翻书"
（搜索相关资料），然后结合资料给出更准确的答案。
生活例子：你问朋友"今天股市怎么样"，
他不是凭记忆猜，而是先打开手机查了一下再告诉你——这就是 RAG。
```

### 案例一：上班族小王的每日早报

**背景**：小王每天早上通勤 40 分钟，想在路上了解天气和行业动态。

**配置方案**：

```
定时任务：每天 07:15 执行

  1. 今日天气（深圳）+ 穿衣建议
```

```
帮我完成今日早报推送到 QQ（QQ号：xxxxxxxxx）：

1. 查询深圳今日天气，给出穿衣建议
2. 搜索今天科技行业最重要的 3 条新闻（来源优先选 36氪、虎嗅），

☀️ 早上好，小王！今日早报

🌤️ 天气：[天气信息]
👔 穿衣：[建议]

📰 科技快讯：
[3条新闻]

📈 A股简报：
[3句话总结]

今天也要加油哦！💪
```

**效果**：小王每天 7:15 准时收到早报，通勤路上 5 分钟看完，对当天的信息心里有数。

# 进阶知识

## 打造更智能的信息助手

### 让龙虾记住你的偏好

每次都要在指令里说明你的偏好很麻烦。用 `AGENTS.md` 文件（参考[第 6 篇](https://qborfy.com/ailearn/openclaw/06.html)的创建方法）把你的偏好固化下来：

```markdown
## 我的信息偏好

### 天气推送

- 城市：深圳
- 格式：简洁版，包含温度、天气、穿衣建议
- 推送渠道：飞书机器人

### 新闻偏好

- 关注领域：AI 技术、互联网行业、A 股
- 不感兴趣：娱乐、体育
- 偏好来源：36 氪、虎嗅、财联社
- 每次推送：5 条，每条 2-3 句话

### 翻译偏好

- 中译英：正式商务语气
- 英译中：口语化，易懂
- 专业术语：保留英文，括号内给中文解释

### 词汇解释风格

- 用大白话，不要太学术
```

### 信息质量的把控技巧

龙虾搜索到的信息不一定 100% 准确，这里有几个提高信息质量的技巧：

**技巧一：指定可信来源**

```
搜索今日 A 股新闻，来源只选：财联社、证券时报、上交所官网
不要使用来源不明的自媒体文章
```

**技巧二：要求标注来源**

```
每条新闻必须标注来源网站和发布时间，
如果找不到明确来源，不要包含在推送里
```

**技巧三：要求区分事实和观点**

```
整理新闻时，如果是分析/观点类内容，
请在开头标注"【观点】"，事实类内容标注"【新闻】"
```

## 常见问题解答

**Q：天气信息不准确怎么办？**

A：在指令里指定数据来源，比如"从中国天气网（weather.com.cn）获取天气数据"，比让龙虾自己找来源更准确。

**Q：飞书机器人发不出消息怎么办？**

A：检查以下几点：

1. Webhook 地址是否正确（复制时有没有多余的空格）
2. 飞书机器人是否还在群里（有时候会被踢出）
3. 飞书技能是否已授权（在龙虾技能管理里查看）

**Q：定时任务没有按时执行怎么办？**

A：检查以下几点：

1. 龙虾是否在运行（定时任务需要龙虾保持运行状态）
2. `scheduler` 技能是否已启用
3. 时区设置是否正确（默认可能是 UTC，需要改成 Asia/Shanghai）

**Q：翻译结果不够准确怎么办？**

A：在指令里加上更多上下文，比如：

```
这是一封商务邮件，对方是我们的美国客户，
请翻译成正式的中文商务语气，
遇到合同相关的专业术语，请保留英文并给出中文解释。
```

# 总结

每日信息助手的核心是**三个字：省时间**。

1. **天气 + 新闻 + 日程**：三件事整合成一条推送，每天早上 5 分钟搞定
2. **查资料 + 翻译 + 解释词汇**：随时触发，不需要打开多个 App 或网站
3. **定时自动化**：配置好之后，龙虾自动执行，你只需要看结果

**从今天开始，试着配置一个最简单的天气推送**——哪怕只有这一个功能，你也会感受到龙虾带来的便利。然后慢慢加上新闻、日程、翻译……一步一步，让龙虾真正融入你的日常。
