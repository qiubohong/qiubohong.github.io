# 是什么

## 为什么龙虾有时候"听不懂"你？

龙虾本质上是一个 AI，它没有人类的常识和默认理解。你说"帮我整理一下"，它不知道：

- 整理**什么**？（文件？邮件？日程？）
- 整理成**什么样**？（按时间？按类型？）
- 整理完**发给谁**？（你自己？还是发到某个地方？）

这就像你让一个刚入职的新员工"帮我整理一下"，他也会一脸懵——不是他不聪明，是你没说清楚。

**好消息是**：给龙虾下指令有规律可循。掌握这套规律，你的指令成功率会从 30% 提升到 90%+。

## 一条好指令的四个要素

记住这个公式：

```
好指令 = 做什么 + 用什么材料 + 输出什么格式 + 发到哪里
```

![好指令四要素对比图](/assets/img/ailearn/openclaw/04/instruction_elements.png)

| 要素           | 说明                     | 举例                             |
| -------------- | ------------------------ | -------------------------------- |
| **做什么**     | 你想让龙虾完成的具体任务 | 整理邮件、查天气、生成报告       |
| **用什么材料** | 龙虾需要处理的内容或来源 | 桌面上的文件、我的邮箱、这个网址 |
| **输出格式**   | 你希望结果长什么样       | 列表、表格、一段话、Word 文档    |
| **发到哪里**   | 结果送到哪里             | 发给我、存到桌面、写入飞书文档   |

### 对比一下：模糊指令 vs 清晰指令

![模糊指令与清晰指令对比图](/assets/img/ailearn/openclaw/04/vague_vs_clear_instruction.png)

**❌ 模糊指令**（龙虾容易出错）：

```
帮我整理一下邮件
```

**✅ 清晰指令**（龙虾能准确执行）：

```
帮我读取今天收到的所有邮件，按"重要"和"一般"两类整理，
用列表格式总结每封邮件的发件人和主题，发给我看。
```

---

**❌ 模糊指令**：

```
帮我写个报告
```

**✅ 清晰指令**：

```
根据桌面上的"销售数据.xlsx"，帮我生成一份本周销售总结报告，
包含：总销售额、环比增长率、销售最好的前3个产品，
用 Word 格式保存到桌面，文件名叫"本周销售总结"。
```

## 三种常用指令类型

![三种指令类型流程图](/assets/img/ailearn/openclaw/04/instruction_types.png)

龙虾能处理的指令，基本上分三大类：

### 类型一：立即执行型

你说一句，龙虾马上去做，做完告诉你结果。

**适合**：临时性的、一次性的任务。

**例子**：

- "帮我查一下北京今天的天气"
- "帮我把桌面上的'合同.pdf'发到我邮箱"
- "帮我搜索一下'Python 入门教程'，把前 5 个链接发给我"

### 类型二：定时执行型

你设定好时间和任务，龙虾到点自动执行，不需要你再说话。

**适合**：重复性的、规律性的任务。

**例子**：

- "每天早上 8 点，把今天的天气和 3 条科技新闻发给我"
- "每周一早上 9 点，提醒我本周有哪些重要日程"
- "每天下午 6 点，把今天的工作文件自动备份到'备份'文件夹"

### 类型三：条件触发型

你设定一个条件，满足条件时龙虾自动执行。

**适合**：需要监控、需要及时响应的任务。

**例子**：

- "如果我的邮箱收到来自'老板@公司.com'的邮件，立刻通知我"
- "如果桌面上出现新的 PDF 文件，自动帮我整理到'文档'文件夹"
- "如果有人在飞书群里@我，把消息内容发到我的私信"

# 怎么做

## 手把手：写出你的第一条好指令

我们用一个真实场景来练习：**让龙虾每天早上给你发早报**。

### 第一步：想清楚你要什么

先回答这四个问题：

1. **做什么**？→ 发早报
2. **用什么材料**？→ 今天的天气 + 今日新闻 + 我的日程
3. **输出格式**？→ 分段落，每部分有标题，简洁易读
4. **发到哪里**？→ 发到我的 QQ（或飞书）

### 第二步：把答案拼成一条指令

```
每天早上 8:00，帮我查询以下内容并整理成早报发给我：
1. 今天北京的天气（温度、天气状况、穿衣建议）
2. 今日科技新闻前3条（标题 + 一句话摘要）
3. 我今天飞书日历上的日程安排

格式要求：
- 每部分有标题（天气 / 新闻 / 日程）
- 内容简洁，不超过200字
- 最后加一句今日鼓励语
```

### 第三步：发给龙虾，看它执行

![指令执行流程图](/assets/img/ailearn/openclaw/04/instruction_execution.png)

把这条指令发给龙虾，它会：

1. 确认收到，告诉你"已设置每日早报，明天早上 8:00 开始执行"
2. 第二天早上 8:00，自动查询天气、新闻、日程
3. 整理好内容，推送到你的 QQ 或飞书

**就这么简单。**

## 让指令更精准的 5 个小技巧

### 技巧一：说"结果"而不是"过程"

❌ "帮我打开浏览器，搜索天气，然后截图发给我"（你在指挥过程）

✅ "告诉我今天北京的天气"（你在要求结果）

龙虾比你更清楚怎么完成任务，你只需要告诉它你想要什么结果。

---

### 技巧二：给出具体的范围

❌ "帮我整理最近的文件"（"最近"是多久？）

✅ "帮我整理桌面上最近 7 天内修改过的文件，按文件类型分类"

---

### 技巧三：说清楚"不要什么"

有时候告诉龙虾"不要做什么"比"要做什么"更重要：

✅ "帮我整理邮件，**不要删除任何邮件**，只做分类标记"

✅ "帮我搜索新闻，**只要中文来源**，不要英文网站"

---

### 技巧四：分步骤下指令（复杂任务）

如果任务很复杂，不要一次说完，分步来：

```
第一步：帮我读取桌面上的"会议录音.mp3"，转成文字。
（等龙虾完成后）
第二步：根据刚才的文字，提取会议中的所有待办事项，整理成列表。
（等龙虾完成后）
第三步：把这个待办列表写入飞书文档，标题叫"本次会议待办"。
```

---

### 技巧五：用"举例"帮龙虾理解格式

如果你对输出格式有特殊要求，直接给一个例子：

```
帮我整理今天的邮件，格式参考下面这个例子：

【重要邮件】
- 发件人：张三 | 主题：项目进度汇报 | 摘要：项目进展顺利，下周交付

【一般邮件】
- 发件人：李四 | 主题：会议通知 | 摘要：周五下午3点开会
```

# 经典案例

## 三个可以直接复制的指令模板

![常用指令模板展示图](/assets/img/ailearn/openclaw/04/instruction_templates.png)

### 模板一：每日早报

```
每天早上 8:00，帮我整理并发送今日早报，内容包括：
1. 【天气】今天[你的城市]的天气，包括温度范围和天气状况
2. 【新闻】今日科技/财经新闻前3条（标题 + 一句话摘要）
3. 【日程】我今天飞书/Google 日历上的所有日程
4. 【鼓励】一句今日励志语

格式简洁，总字数不超过300字，发到我的飞书私信。
```

---

### 模板二：文件自动整理

```
帮我整理桌面上的文件，规则如下：
- 图片文件（.jpg/.png/.gif）→ 移动到桌面的"图片"文件夹
- 文档文件（.doc/.docx/.pdf）→ 移动到桌面的"文档"文件夹
- 压缩文件（.zip/.rar）→ 移动到桌面的"压缩包"文件夹
- 其他文件 → 不动

注意：不要删除任何文件，只做移动操作。
整理完成后告诉我移动了多少个文件。
```

---

### 模板三：会议纪要生成

```
帮我处理这个会议录音文件（文件路径：[你的文件路径]）：
1. 先把录音转成文字
2. 从文字中提取：
   - 会议主要讨论内容（3-5条）
   - 会议决定事项（列表）
   - 待办事项和负责人（如果有提到）
3. 整理成会议纪要，格式如下：
   - 会议时间：[自动识别或填写今天日期]
   - 参会人员：[从录音中识别]
   - 主要内容：
   - 决定事项：
   - 待办事项：
4. 保存为 Word 文档，文件名"[今天日期]会议纪要"，存到桌面
```

---

### 模板四：邮件自动回复助手

```
帮我监控我的邮箱，规则如下：
- 如果收到来自[重要联系人邮箱]的邮件，立刻通过飞书通知我，并附上邮件摘要
- 如果收到标题包含"紧急"或"重要"的邮件，同样立刻通知我
- 其他邮件，每天下午6点汇总发给我一次

注意：只通知，不要自动回复任何邮件。
```

# 动手试试！

## 现在就能做的三件事

### 第一件：发一条"立即执行"的指令

打开你的 QQ 或飞书，找到龙虾机器人，发送这条指令：

```
帮我查一下今天[你的城市]的天气，用一句话告诉我今天适不适合出门。
```

看看龙虾的回复是否准确。如果回复不够好，试着加上更多细节再发一次。

---

### 第二件：设置你的第一个定时任务

把上面的"每日早报"模板复制过去，把 `[你的城市]` 改成你所在的城市，发给龙虾。

看看明天早上 8 点，龙虾会不会主动给你发早报。

---

### 第三件：练习"改进指令"

![指令改进迭代流程图](/assets/img/ailearn/openclaw/04/instruction_iteration.png)

如果龙虾的回复不是你想要的，不要放弃，试着这样说：

```
你刚才的回复不太对，我想要的是[描述你期望的结果]，
能重新做一次吗？
```

龙虾会根据你的反馈重新执行。**指令是可以迭代的**，第一次不完美很正常。

---

# 进阶知识

## 指令失败了怎么办？

遇到龙虾"没听懂"或"做错了"，按这个顺序排查：

**第一步：看错误提示**

- 如果龙虾明确告诉你哪里错了（比如"找不到这个文件"），就按提示修改

**第二步：检查龙虾是否理解**

- 如果龙虾没理解你的意思，重新说一遍，把"做什么+用什么材料+输出什么格式+发到哪里"都说清楚

**第三步：用例子示范**

- 如果龙虾理解了但做出来的不是你想要的，直接给一个例子，告诉它你希望的结果长什么样

**第四步：重新发送指令**

- 修改后重新发送，看看这次对不对

**第五步：如果还不对，拆分成小任务**

- 把大任务拆成几个小步骤，一步一步来

**记住：指令是可以反复调整的**，第一次不完美很正常，多试几次就熟练了。

## 龙虾处理任务的顺序规则

当你给龙虾设置了多个任务，它会按这个顺序处理：

**第一优先：你直接说的话**

- 你随时发的指令，龙虾会立刻响应

**第二优先：有条件的自动任务**

- 比如"收到老板邮件就通知我"，这种任务比定时任务优先

**第三优先：定时任务**

- 比如"每天早上 8 点发早报"，到点执行，但如果你在说话，先处理你

**最后：没有时间要求的任务**

- 龙虾空闲时再处理这些任务

## 建立你的"常用指令库"

随着使用时间增长，你会积累很多好用的指令。建议你在电脑上建一个简单的文档，把常用的指令保存下来：

**具体做法**：

1. 在桌面新建一个文本文档，名字叫"我的龙虾常用指令"
2. 把好用的指令分类保存，比如：
   - 每日早报指令
   - 文件整理指令
   - 会议纪要指令
3. 下次需要用的时候，直接打开文档复制粘贴

**好处**：不用每次都重新想指令，省时省力

# 总结

学会给龙虾下指令，只需要记住三件事：

1. **好指令的公式**：做什么 + 用什么材料 + 输出什么格式 + 发到哪里。四个要素说清楚，龙虾基本不会出错。

2. **三种指令类型**：立即执行（临时任务）、定时执行（重复任务）、条件触发（监控任务）。根据你的需求选对类型。

3. **指令可以迭代**：第一次不完美很正常，根据龙虾的回复调整指令，越用越顺手。
