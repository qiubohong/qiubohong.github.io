# 是什么

## 安装龙虾的五种方式

OpenClaw 提供了五种安装方式，就像买手机有不同的套餐一样：

![OpenClaw安装方式对比图：展示一键脚本、npm安装、苹果客户端三种方式的优缺点对比](/assets/img/ailearn/openclaw/03/installation_comparison.png)

**新手套餐：一键脚本安装**（最快上手，5 分钟搞定）

- 适合：完全不懂技术的纯新手
- 就像：买手机直接开机就能用
- 优点：自动完成所有配置，无需手动操作
- 缺点：依赖网络环境

**技术爱好者套餐：npm 安装**（最灵活）

- 适合：有一定技术基础，想深度定制
- 就像：自己组装电脑，想怎么配就怎么配
- 优点：完全控制配置，随时更新
- 缺点：需要懂一点命令行

**苹果用户套餐：客户端安装**（最省心）

- 适合：使用苹果电脑的用户
- 就像：在 App Store 下载应用
- 优点：图形界面操作，无需命令行
- 缺点：仅支持 macOS

**云服务器套餐：腾讯云一键秒部署**（服务器用户首选）

- 适合：想快速搭建自己的机器人服务
- 就像：租一间精装修的房子，拎包入住
- 优点：云端部署，24 小时运行，无需本地电脑
- 缺点：需要付费购买云服务器

**云服务器套餐：阿里云快速部署**（企业用户推荐）

- 适合：企业级应用，需要稳定服务
- 就像：买精装房，配套齐全
- 优点：企业级支持，稳定可靠
- 缺点：成本相对较高

**本篇重点讲一键脚本**，因为这是新手最快上手的方式。其他四种方式会在文末简要说明。

# 怎么做

## 准备工作

在开始安装前，确保你的电脑满足以下条件：

- **操作系统**：Windows 10/11、macOS 10.15+、Linux Ubuntu 18.04+
- **内存**：至少 8GB（推荐 16GB）
- **硬盘空间**：至少 2GB 可用空间
- **网络**：能正常访问 GitHub 和国内大模型平台

### 如何打开"黑窗口"（终端/命令提示符）

很多新手一听到"命令行"就害怕，其实没那么复杂。终端就是一个可以输入命令的窗口，就像你平时用微信打字一样简单。

#### Windows 用户

**方法一：使用命令提示符**

1. 按键盘上的 `Win + R` 键（Win 键就是键盘左下角那个 Windows 图标键）
2. 在弹出的窗口中输入 `cmd`
3. 按回车键，会弹出一个黑色的窗口，这就是命令提示符

**方法二：使用 PowerShell（推荐）**

1. 在 Windows 搜索栏输入 `PowerShell`
2. 点击"Windows PowerShell"打开
3. PowerShell 比命令提示符功能更强大

#### macOS 用户

**方法一：使用 Spotlight 搜索**

1. 按键盘上的 `Cmd + 空格` 键（Cmd 键就是键盘左下角的 Command 键，上面有个 ⌘ 符号）
2. 在搜索框中输入"终端"或"Terminal"
3. 按回车键，会弹出一个白色的窗口

**方法二：从启动台打开**

1. 点击屏幕底部的"启动台"图标（那个有九个小方块的图标）
2. 找到"终端"应用并点击打开

#### Linux 用户

- **Ubuntu/Debian**：按 `Ctrl + Alt + T` 直接打开终端
- **其他发行版**：在应用菜单中搜索"终端"或"Terminal"

> **小贴士**：终端窗口打开后，你会看到一个闪烁的光标，那意味着电脑在等着你输入命令。

### 检查 Node.js 环境（仅 npm 安装需要）

如果你选择**npm 安装方式**，需要先检查电脑是否安装了 Node.js。

**如何检查是否已安装 Node.js**：

在终端中输入以下命令并按回车：

```bash
node -v
```

**可能的结果**：

- **如果看到版本号**：比如 `v18.17.0`，说明已经安装了 Node.js，可以跳过安装步骤
- **如果提示"命令不存在"**：说明没有安装，需要先安装 Node.js

#### 如何安装 Node.js

**Windows 用户**：

1. 访问 [Node.js 官网](https://nodejs.org)
2. 点击"下载"按钮（建议选择 LTS 版本，长期支持版本更稳定）
3. 下载完成后，双击安装包
4. 一直点"下一步"，使用默认设置即可
5. 安装完成后，重新打开终端，再次输入 `node -v` 验证

**macOS 用户**：

**方法一：官网安装**

1. 访问 [Node.js 官网](https://nodejs.org)
2. 下载 macOS 版本的安装包（.pkg 文件）
3. 双击安装包，按照提示完成安装
4. 安装完成后，重新打开终端，输入 `node -v` 验证

**方法二：使用 Homebrew（推荐，更方便）**
如果你已经安装了 Homebrew（一个 Mac 上常用的软件管理工具），可以用这个命令：

```bash
brew install node
```

**Linux 用户**：

**Ubuntu/Debian**：

```bash
sudo apt update
sudo apt install nodejs npm
```

**CentOS/RHEL**：

```bash
sudo yum install nodejs npm
```

> **为什么要安装 Node.js？** Node.js 是一个可以让 JavaScript 在电脑上运行的环境，OpenClaw 的 npm 安装包需要它才能正常工作。

**特别提醒**：如果你在第 2 篇注册了智谱 AI，请把 API Key 准备好。如果还没注册，现在去注册：[智谱 AI 开放平台](https://open.bigmodel.cn)

## 第一步：一键脚本安装（新手首选）

这是**最简单**的安装方式，全程只需要复制粘贴一条命令。

### 1. 打开"黑窗口"（终端/命令提示符）

- **Windows**：按 `Win + R`，输入 `cmd`，回车
- **macOS**：按 `Cmd + 空格`，搜索"终端"，打开
- **Linux**：按 `Ctrl + Alt + T` 打开终端

> **什么是终端？** 你可以把它理解成一个"打字机窗口"，在这里输入命令，电脑就会执行。

### 2. 执行一键安装命令

在终端中复制粘贴以下命令：

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

**这条命令的作用**：

- `curl`：从网上下载文件
- `https://openclaw.ai/install.sh`：官方安装脚本地址
- `| bash`：把下载的内容直接运行

**安全提醒**：这个脚本来自 OpenClaw 官方，可以放心使用。

### 3. 等待安装完成

安装过程大约需要 5-10 分钟，取决于你的网速。脚本会自动：

1. 检查你的系统环境
2. 下载最新版本的 OpenClaw
3. 安装必要的依赖包
4. 配置运行环境

你会看到类似这样的进度：

![OpenClaw正在安装中](https://mintcdn.com/clawdhub/U8jr7qEbUc9OU9YR/assets/install-script.svg?w=2500&fit=max&auto=format&n=U8jr7qEbUc9OU9YR&q=85&s=c602d6faf7d925191299469008e0d57b)

**如果中途卡住**：可能是网络问题，耐心等待或重新执行命令。

### 4. 输入"大脑钥匙"（API Key）

安装完成后，系统会提示你输入大模型配置。按照提示填写：

```
请选择大模型平台（输入数字）：
1. 智谱 AI (GLM)
2. 通义千问 (Qwen)
3. MiniMax
4. DeepSeek

请输入你的选择（默认 1）：1
请输入你的 API Key：zhipuai-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

把你在第 2 篇拿到的智谱 AI API Key 粘贴进去。

> **API Key 是什么？** 就像你家门的钥匙，有了这把钥匙，龙虾才能调用大模型的思考能力。

### 5. 启动龙虾（三步完成）

安装完成后，需要通过三个命令来启动龙虾。不要担心，这三个命令很简单：

**第一步：运行配置向导**

在终端中输入：

```bash
openclaw onboard
```

这个命令会启动一个交互式配置向导，引导你完成以下设置：

1. **风险确认**：提示你 OpenClaw 会访问你的消息和应用，需要你确认是否同意
2. **上船模式选择**：选择你想要的运行模式（如 Web 模式、消息机器人模式等）
3. **大模型/提供商配置**：选择使用哪个大模型平台（智谱 AI、通义千问等）
4. **渠道选择**：选择要接入的消息渠道（QQ、飞书、企业微信等）
5. **技能配置**：选择要启用的技能（天气、日历、新闻等）

按照向导提示一步步操作，就像填表一样简单。

**第二步：启动网关服务**

配置完成后，启动 OpenClaw 的网关服务：

```bash
openclaw gateway start
```

这个命令会启动 OpenClaw 的核心服务，包括：

- Web 服务（提供 Web 界面）
- 消息接收器（接收 QQ、飞书等消息）
- 任务调度器（执行定时任务）
- 技能引擎（调用各种技能）

你会看到类似这样的输出：

```
[INFO] OpenClaw Gateway 启动中...
[INFO] Web 服务已启动：http://127.0.0.1:18789
[INFO] QQ 机器人已连接
[INFO] 飞书机器人已连接
[INFO] 任务调度器已启动
[INFO] OpenClaw 服务运行中 ✓
```

> **什么是网关？** 网关就像一个"总机"，负责接收所有消息，然后分发给不同的处理器处理。

**第三步：打开 Web 界面（可选）**

如果你想在浏览器中管理 OpenClaw，可以打开 Web 界面：

```bash
openclaw dashboard
```

这个命令会自动在浏览器中打开 OpenClaw 的管理界面，地址是：

```
http://127.0.0.1:18789
```

在 Web 界面中，你可以：

- 查看运行状态
- 管理技能
- 查看日志
- 配置参数

> **注意**：`openclaw dashboard` 命令会自动打开浏览器。如果你的浏览器没有自动打开，可以手动在浏览器中输入 `http://127.0.0.1:18789` 访问。

**恭喜！** 龙虾已经在你电脑上运行起来了。

> **三步启动流程总结**：
>
> 1. `openclaw onboard` - 配置向导，设置参数
> 2. `openclaw gateway start` - 启动服务
> 3. `openclaw dashboard` - 打开管理界面（可选）

## 其他四种安装方式（简要说明）

### npm 安装方式（适合有技术基础的用户）

如果你熟悉命令行和 npm，可以用 npm 安装：

```bash
# 1. 确保已安装 Node.js（参考上面的安装教程）
# 验证是否安装成功
node -v

# 2. 安装最新版本的 OpenClaw
npm install -g openclaw@latest

# 3. 运行配置向导
openclaw onboard

# 4. 启动网关服务
openclaw gateway start

# 5. 打开 Web 管理界面（可选）
openclaw dashboard
```

**适合人群**：

- 熟悉命令行操作
- 了解 Node.js 和 npm
- 希望随时更新到最新版本
- 想要深度定制配置

**优点**：

- 可以随时更新到最新版本
- 完全控制安装位置和配置
- 适合开发者和高级用户

**缺点**：

- 需要懂一点技术
- 需要手动配置环境变量
- 遇到问题需要自己排查

**注意**：

- npm 安装的包名是 `openclaw`，不是 `@openclaw/cli`
- `@latest` 表示安装最新版本，如果不加这个标签，可能会安装旧版本
- 安装完成后，启动流程和一键脚本一样，需要运行 `openclaw onboard` 和 `openclaw gateway start`

# 常见问题解答

## Q1：一键脚本安装失败怎么办？

**A**：常见原因和解决方案：

- **网络问题**：脚本需要从 GitHub 下载文件，如果网络不稳定会失败
  - 解决方案：重新执行命令，或尝试使用 VPN
- **权限问题**：脚本需要权限来安装软件
  - 解决方案：在命令前加 `sudo`：`curl -fsSL https://openclaw.ai/install.sh | sudo bash`
- **系统不兼容**：脚本可能不支持你的系统版本
  - 解决方案：尝试其他安装方式（npm 或客户端）

## Q2：npm 安装时找不到命令怎么办？

**A**：确保你按照正确的步骤操作：

**第一步：确认 Node.js 已安装**

```bash
node -v
```

如果提示"命令不存在"，请先按照上面的教程安装 Node.js。

**第二步：使用正确的命令安装**

```bash
# 正确的命令（添加 @latest 确保安装最新版本）
npm install -g openclaw@latest

# 错误的命令（不要用这个）
npm install -g @openclaw/cli
```

**第三步：验证安装是否成功**

```bash
openclaw -v
```

如果显示版本号，说明安装成功。

**第四步：按照三步流程启动**

```bash
# 1. 运行配置向导
openclaw onboard

# 2. 启动网关服务
openclaw gateway start

# 3. 打开 Web 界面（可选）
openclaw dashboard
```

**如果遇到权限问题**（尤其是 macOS 和 Linux 用户）：

```bash
# 使用 sudo 安装
sudo npm install -g openclaw@latest
```

> **注意**：使用 sudo 安装时，系统会要求你输入管理员密码。

## Q3：苹果客户端无法安装怎么办？

**A**：常见问题：

- **无法打开应用**：macOS 安全设置阻止了未签名的应用
  - 解决方案：在"系统偏好设置"→"安全性与隐私"中允许打开
- **找不到下载链接**：访问 [GitHub 发布页面](https://github.com/openclaw/openclaw/releases/latest)
  - 下载文件名类似 `OpenClaw-v1.0.0.dmg` 的文件

## Q4：如何更新龙虾到最新版本？

**A**：根据你的安装方式选择更新方法：

- **一键脚本用户**：重新执行安装命令，脚本会自动更新
  ```bash
  curl -fsSL https://openclaw.ai/install.sh | bash
  ```
- **npm 用户**：
  ```bash
  npm update -g openclaw
  ```
- **苹果客户端用户**：在应用内检查更新，或重新下载最新版本

## Q5：启动命令 `openclaw start` 提示不存在怎么办？

**A**：OpenClaw 没有直接的 `openclaw start` 命令，需要按照三步流程启动：

**正确的启动流程**：

```bash
# 第一步：运行配置向导（首次安装必须执行）
openclaw onboard

# 第二步：启动网关服务（每次启动都要执行）
openclaw gateway start

# 第三步：打开 Web 界面（可选）
openclaw dashboard
```

**为什么需要三步？**

- `openclaw onboard`：配置向导，只需要在首次安装或需要重新配置时运行
- `openclaw gateway start`：启动核心服务，每次重启电脑后都需要运行
- `openclaw dashboard`：打开 Web 管理界面，可选操作

**如果想让服务开机自动启动**：

可以使用系统的开机启动项功能，或者在配置文件中设置自动启动。具体方法请参考官方文档。
