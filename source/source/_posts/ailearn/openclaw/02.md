上一篇我们搞清楚了 OpenClaw 是什么。这一篇解决一个很多人卡住的问题：**龙虾需要一个"大脑"（大模型）才能思考，这个大脑要花钱吗？**

答案是：**完全不用花钱。** 国内有好几家大模型平台提供免费额度，注册就能用，足够日常使用。

<!-- more -->

# 是什么

## 为什么龙虾需要"大脑"？

OpenClaw 本身只负责接收你的指令、调用工具、执行任务——相当于一双勤快的手。但它自己不会"思考"，需要借助大模型来理解你说的话、制定计划、决定下一步怎么做。

你可以把它理解成这样：

```
龙虾 = 勤快的手脚（执行）+ 借来的大脑（大模型）
```

没有大脑，龙虾就是一个空壳，什么都做不了。有了大脑，它才能理解"帮我整理文件"这句话背后的意思，并一步步把事情做完。

## 大模型 API 是什么？

大模型平台（比如智谱、通义千问）提供了一种接口，叫做 **API**（你可以理解成"一扇门"）。

- 你注册账号后，平台会给你一把**钥匙**（叫做 API Key）
- 龙虾拿着这把钥匙，就能通过这扇门调用大模型的思考能力
- 每次调用会消耗一点"额度"（就像话费），平台会免费送你一批额度

**本篇重点讲智谱 AI 的注册和配置**，因为它额度最大、配置最简单，是新手首选。其他平台的注册流程大同小异，后面会简要说明。

# 怎么做

## 第一步：注册智谱 AI，领取免费额度

**1. 打开注册页面**

访问 [https://open.bigmodel.cn](https://open.bigmodel.cn)，点击右上角"注册"。

**2. 填写信息**

用手机号注册，填写验证码，设置密码，完成。整个过程不超过 2 分钟。

**3. 实名认证（可选但推荐）**

完成实名认证后，免费额度会从 100 万 tokens 提升到 **1000 万 tokens**，强烈建议做一下，只需要身份证号和手机号。

**4. 确认免费额度**

注册完成后，进入"控制台" → "资源包"，可以看到你的免费额度。

---

## 第二步：获取 API Key（你的"钥匙"）

**1. 进入 API Key 管理页面**

登录后，点击右上角头像 → "API Keys" → "新建 API Key"。

**2. 给这把钥匙起个名字**

随便填，比如"龙虾专用"，方便你以后区分。

**3. 复制并保存 API Key**

系统会生成一串字符，格式类似：

```
zhipuai-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

**⚠️ 重要提示**：这串字符只显示一次，请立刻复制，粘贴到记事本或备忘录里保存好。如果忘记了，只能重新生成一个新的。

---

## 第三步：填入龙虾配置

拿到 API Key 之后，在安装龙虾时（第 3 篇会详细讲），你需要在配置文件里填入以下信息：

```yaml
# 龙虾配置文件示例（第 3 篇会详细讲怎么填）
model_provider: zhipuai # 大模型平台：智谱 AI
model_name: glm-4-flash # 使用的模型名称（免费版）
api_key: 你的API Key # 粘贴你刚才复制的那串字符
```

> **glm-4-flash 是什么？** 这是智谱 AI 的免费模型，速度快、能力强，完全够日常使用。如果你想要更强的能力，可以选 glm-4（收费，但很便宜）。

---

## 其他平台的注册方式（简要说明）

### 通义千问（阿里云）

1. 访问 [https://dashscope.aliyun.com](https://dashscope.aliyun.com)
2. 用支付宝或淘宝账号登录（无需重新注册）
3. 开通"灵积模型服务"（这是阿里云对大模型 API 的叫法，开通后才能使用），免费领取额度
4. 在"API Key 管理"页面创建 Key

配置时填写：

```yaml
model_provider: qwen
model_name: qwen-turbo # 免费版
api_key: 你的API Key
```

### MiniMax

1. 访问 [https://api.minimax.chat](https://api.minimax.chat)
2. 手机号注册，完成实名认证
3. 在"账户信息"页面找到 API Key 和 Group ID（Group ID 是你账号的唯一编号，MiniMax 需要两个一起用才能正常工作）

配置时填写：

```yaml
model_provider: minimax
model_name: abab6.5s-chat # 免费版
api_key: 你的API Key
group_id: 你的Group ID
```

### DeepSeek

1. 访问 [https://platform.deepseek.com](https://platform.deepseek.com)
2. 手机号注册
3. 在"API Keys"页面创建 Key

配置时填写：

```yaml
model_provider: deepseek
model_name: deepseek-chat # 通用版
api_key: 你的API Key
```

**预计时间**：5 分钟。

---

### 第二件：测试你的 API Key 是否有效

注册完成后，可以用下面这个简单的方法测试 API Key 是否能正常使用。

打开电脑上的"终端"（Mac）或"命令提示符"（Windows），输入以下命令（把 `你的API Key` 替换成你刚才复制的那串字符）：

```bash
curl https://open.bigmodel.cn/api/paas/v4/chat/completions \
  -H "Authorization: Bearer 你的API Key" \
  -H "Content-Type: application/json" \
  -d '{"model":"glm-4-flash","messages":[{"role":"user","content":"你好"}]}'
```

如果屏幕上出现了一大段文字，里面能找到"你好"的回复内容，说明 API Key 有效，可以正常使用。

> **看不懂命令行？没关系。** 这一步是可选的，跳过也完全没问题。第 3 篇安装龙虾时，会有更简单的方式验证配置是否正确。

---

### 第三件：备用方案——多注册一个平台

建议同时注册**智谱 AI** 和 **DeepSeek** 两个平台，原因是：

- 智谱额度大，适合日常高频使用
- DeepSeek 推理能力强，适合复杂任务
- 两个都是免费的，多一个备用没有坏处

如果某个平台临时出现故障或额度用完，可以随时切换到另一个，不影响龙虾正常工作。

# 进阶知识

## tokens 到底是什么？怎么估算用量？

很多人看到"1000 万 tokens"不知道是多还是少，这里给你一个直观的感受：

![tokens用量估算图表：展示不同任务类型的tokens消耗和1000万tokens的可执行次数对比](/assets/img/ailearn/openclaw/02/tokens_usage.png)

对于普通人来说，每天用龙虾处理 10-20 个任务，1000 万 tokens 大约能用 **3-6 个月**。

## 如果免费额度用完了怎么办？

有三个选择：

**选项一：等下个月刷新**

大部分平台的免费额度是每月重置的，等到下个月 1 号就会重新有额度。

**选项二：充值（非常便宜）**

如果你用量比较大，可以选择充值。以智谱 AI 为例，glm-4-flash 模型的价格是 **0.001 元/千 tokens**（也就是 1 块钱能用 100 万字），日常使用基本上一个月花不了 5 块钱。

**选项三：切换到其他平台**

你注册了多个平台，哪个有额度就用哪个，在龙虾配置里改一下 API Key 就行，5 分钟搞定。

## 大模型的"上下文长度"是什么意思？

你可能会看到大模型介绍里写"支持 128K 上下文"，这是什么意思？

简单理解：**上下文长度 = 龙虾一次能"记住"多少内容**。

- 上下文越长，龙虾处理大文件（比如长文档、长对话）的能力越强
- 128K 上下文大约能处理 10 万字的内容，相当于一本中等厚度的书

对于日常任务，32K 上下文（约 2.4 万字）已经完全够用。如果你需要处理超长文档，选择支持 128K 的模型（智谱 glm-4-long 支持 128K，且有免费额度）。

# 总结

这一篇解决了"龙虾的大脑从哪来"的问题，记住三件事：

1. **龙虾需要大模型 API 才能思考**：API Key 就是一把钥匙，让龙虾能调用大模型的能力。

2. **国内有大量免费额度可用**：智谱 AI 每月 1000 万 tokens，注册就有，完全够日常使用。

3. **新手首选智谱 AI**：注册简单、额度最大、配置最方便，是目前最适合新手的选择。
