今天我们来学习 **AI Agent 的设计模式 —— 多智能体协作 (Multi-agent Collaboration)**

> 一句话核心：**多智能体协作**（Multi-agent Collaboration）是让多个专业 AI Agent 像一个高效团队一样协同工作，将复杂任务分解成子任务，由各领域的专家 Agent 并行处理，再整合成最终结果。

说实话，这种模式太像我们日常工作中团队协作的样子了。想想看，做一个项目需要多少人？产品经理出需求、设计师画图、前端写页面、后端写接口、测试找 BUG。每个人都是某个领域的专家，但又需要紧密配合。

单个 AI Agent 再强大，也有它的局限。它就像一个超级全栈工程师，虽然什么都懂，但处理复杂任务时还是会力不从心——速度慢、质量不稳定、难以覆盖所有专业领域。

多智能体协作的核心价值在于**分工协作和并行处理**——把一个大问题拆成多个小问题，交给不同的专家 Agent 同步处理，最后整合结果。就像组建了一个 AI 梦之队！

# 是什么

老规矩，先看图解：

![多智能体协作工作原理](/assets/img/ailearn/daily/36/1.png)

从图里能看到，多智能体协作模式有四个核心组件：

- **Coordinator/Hub（协调中心）**：团队总指挥，负责任务分解、分配和结果整合
- **Specialized Agents（专业 Agent）**：各个领域的专家，如研究 Agent、分析 Agent、写作 Agent、审核 Agent
- **Shared Memory（共享内存）**：团队成员之间的"群聊记录"，确保信息同步
- **Communication Channel（通信通道）**：Agent 之间交换消息的网络

整个协作流程是这样的：

1. **接收复杂任务**：用户说"帮我写一篇关于 AI 发展趋势的深度报告"
2. **任务分解**：Coordinator 分析任务，拆成：资料收集 → 数据分析 → 内容撰写 → 质量审核
3. **并行分配**：把子任务分配给对应的专家 Agent
4. **协同执行**：各 Agent 同步工作，通过共享内存交流进展
5. **结果汇总**：Coordinator 收集各 Agent 的结果，整合成最终报告
6. **输出交付**：完整的报告返回给用户

举个我折腾的真实例子：

之前我尝试让单个 Agent 写一篇技术调研报告，结果你懂的——它写是写出来了，但深度不够，数据也不够准确，就像一个人熬夜赶出来的作业。

后来改用多智能体协作：

- **Research Agent**：先花半小时搜集最新的论文和行业数据
- **Analysis Agent**：分析数据，提取关键趋势和洞察
- **Writing Agent**：根据研究材料撰写报告内容
- **Review Agent**：审核内容准确性，检查数据和引用
- **Coordinator**：把控整体进度，确保各部分衔接流畅

效果完全不一样！报告的质量、深度、准确性都大幅提升。这就是多智能体协作的威力：**众人拾柴火焰高**。

## 核心架构

多智能体协作的核心代码结构：

```python
class MultiAgentCollaboration:
    def __init__(self):
        self.coordinator = CoordinatorAgent()  # 协调中心
        self.agents = {
            "research": ResearchAgent(),
            "analysis": AnalysisAgent(),
            "writing": WritingAgent(),
            "review": ReviewAgent()
        }
        self.shared_memory = SharedMemory()  # 共享内存

    def execute(self, task):
        # Step 1: Coordinator分解任务
        subtasks = self.coordinator.decompose(task)

        # Step 2: 并行分配给各Agent
        results = self.parallel_execute(subtasks)

        # Step 3: Coordinator整合结果
        final_output = self.coordinator.integrate(results)

        return final_output
```

关键点在于：

- **明确的角色分工**：每个 Agent 有清晰的责任边界
- **共享状态**：所有 Agent 可以访问共享内存，了解全局进展
- **并行执行**：独立子任务同时处理，提升效率
- **灵活协作**：Agent 之间可以直接通信，解决依赖问题
- **结果整合**：Coordinator 负责把各部分汇集成完整答案

```python
from typing import Dict, List, Optional, Callable, Any
from dataclasses import dataclass, field
from enum import Enum
from concurrent.futures import ThreadPoolExecutor, as_completed
import json
import time
from datetime import datetime

# 模拟 LLM 客户端
class MockLLM:
    def generate(self, prompt: str, system_prompt: str = "") -> str:
        """模拟 LLM 生成响应"""
        return f"[LLM] 基于以下输入分析：{prompt[:60]}..."

llm = MockLLM()


@dataclass
class Task:
    """任务定义"""
    id: str
    type: str  # 任务类型
    description: str  # 任务描述
    dependencies: List[str] = field(default_factory=list)  # 依赖的其他任务ID
    input_data: Dict = field(default_factory=dict)  # 输入数据
    output_data: Any = None  # 输出数据
    status: str = "pending"  # pending, running, completed, failed
    assigned_to: str = ""  # 分配给哪个Agent


@dataclass
class Message:
    """Agent间通信消息"""
    sender: str
    receiver: str  # "all" 或特定Agent名称
    content: str
    timestamp: float = field(default_factory=time.time)
    message_type: str = "info"  # info, request, response, result


class SharedMemory:
    """
    共享内存：所有Agent都可以访问的公共空间
    相当于团队的"群聊记录"和"共享文档"
    """

    def __init__(self):
        self.data: Dict[str, Any] = {}  # 共享数据存储
        self.messages: List[Message] = []  # 消息历史
        self.task_results: Dict[str, Any] = {}  # 任务结果存储

    def write(self, key: str, value: Any, agent_name: str = "system"):
        """Agent写入共享数据"""
        self.data[key] = {
            "value": value,
            "updated_by": agent_name,
            "timestamp": datetime.now().isoformat()
        }
        print(f"  📝 [{agent_name}] 写入共享内存: {key}")

    def read(self, key: str) -> Any:
        """读取共享数据"""
        entry = self.data.get(key)
        return entry["value"] if entry else None

    def send_message(self, message: Message):
        """发送消息"""
        self.messages.append(message)
        receiver_str = message.receiver if message.receiver != "all" else "所有人"
        print(f"  📨 [{message.sender}] → [{receiver_str}]: {message.content[:50]}...")

    def get_messages_for(self, agent_name: str) -> List[Message]:
        """获取某个Agent的相关消息"""
        return [m for m in self.messages
                if m.receiver == agent_name or m.receiver == "all" or m.sender == agent_name]

    def store_task_result(self, task_id: str, result: Any):
        """存储任务结果"""
        self.task_results[task_id] = result

    def get_task_result(self, task_id: str) -> Any:
        """获取任务结果"""
        return self.task_results.get(task_id)


class BaseAgent:
    """
    Agent基类：所有专业Agent的父类
    """

    def __init__(self, name: str, role: str, skills: List[str]):
        self.name = name
        self.role = role
        self.skills = skills
        self.shared_memory: Optional[SharedMemory] = None
        self.coordinator: Optional['CoordinatorAgent'] = None

    def connect(self, shared_memory: SharedMemory, coordinator: 'CoordinatorAgent'):
        """连接到共享内存和协调中心"""
        self.shared_memory = shared_memory
        self.coordinator = coordinator

    def execute(self, task: Task) -> Any:
        """执行任务的模板方法（子类必须实现具体逻辑）"""
        raise NotImplementedError

    def send_message(self, receiver: str, content: str, message_type: str = "info"):
        """向其他Agent发送消息"""
        if self.shared_memory:
            msg = Message(
                sender=self.name,
                receiver=receiver,
                content=content,
                message_type=message_type
            )
            self.shared_memory.send_message(msg)

    def _call_llm(self, prompt: str) -> str:
        """调用LLM"""
        system_prompt = f"你是{self.name}，角色是{self.role}。专长：{', '.join(self.skills)}"
        return llm.generate(prompt, system_prompt)

class AnalysisAgent(BaseAgent):
    """
    分析Agent：负责分析数据，提取洞察
    """

    def __init__(self):
        super().__init__(
            name="分析专家",
            role="数据分析师",
            skills=["数据挖掘", "趋势分析", "洞察提取", "可视化"]
        )

    def execute(self, task: Task) -> Any:
        """执行分析任务"""
        print(f"\n  📊 [{self.name}] 开始执行分析任务: {task.id}")

        # 检查依赖的研究数据是否就绪
        dependency_id = task.input_data.get("depends_on")
        if dependency_id:
            # 等待研究数据
            max_wait = 10
            waited = 0
            while waited < max_wait:
                research_data = self.shared_memory.get_task_result(dependency_id)
                if research_data:
                    break
                time.sleep(0.5)
                waited += 0.5

        research_data = self.shared_memory.read(f"research_{dependency_id}") if dependency_id else None

        if not research_data:
            print(f"  ⚠️ [{self.name}] 未找到研究数据，使用默认分析")
            research_data = "[默认研究数据]"

        # 生成分析结果
        analysis_result = f"""
【分析维度】市场趋势、技术突破、竞争格局、投资机会
【核心洞察】
1. 市场趋势：行业处于高速增长期，年复合增长率25%+
2. 技术突破：多模态模型和Agent技术成为新热点

4. 投资机会：垂直领域应用和基础设施层值得关注
【数据支撑】基于{len(research_data)}字的研究材料进行深度分析
【可视化建议】建议制作趋势图、市场份额饼图、技术路线图"""

class ReviewAgent(BaseAgent):
    """
    审核Agent：负责质量把控
    """

    def __init__(self):
        super().__init__(
            name="审核专家",
            role="质量审核员",
            skills=["内容审核", "质量检查", "事实核查", "逻辑审查"]
        )

    def execute(self, task: Task) -> Any:
        """执行审核任务"""
        print(f"\n  ✅ [{self.name}] 开始执行审核任务: {task.id}")

        # 获取报告
        dependency_id = task.input_data.get("depends_on")
        report = self.shared_memory.read(f"report_{dependency_id}") if dependency_id else None

        if not report:
            return {"approved": False, "feedback": "未找到报告内容"}

        # 模拟审核检查
        checks = [
            ("数据准确性", True, "数据来源可靠"),
            ("逻辑连贯性", True, "论证逻辑清晰"),
            ("内容完整性", True, "覆盖必要章节"),
            ("格式规范性", True, "符合报告标准"),
        ]

        feedback = []
        for check_name, passed, comment in checks:
            status = "✓" if passed else "✗"
            feedback.append(f"{status} {check_name}: {comment}")

        review_result = {
            "approved": all(passed for _, passed, _ in checks),
            "feedback": "\n".join(feedback),
            "suggestions": "建议：可增加更多图表支持",
            "score": 85
        }

        self.shared_memory.write(f"review_{task.id}", review_result, self.name)

        print(f"  ✅ [{self.name}] 审核完成，评分: {review_result['score']}")
        return review_result


class CoordinatorAgent:
    """
    协调中心：多智能体系统的总指挥
    """

    def __init__(self):
        self.name = "协调中心"
        self.shared_memory = SharedMemory()
        self.agents: Dict[str, BaseAgent] = {}
        self.task_counter = 0

    def register_agent(self, agent: BaseAgent):
        """注册Agent到系统"""
        agent.connect(self.shared_memory, self)
        self.agents[agent.name] = agent
        print(f"  📝 注册Agent: {agent.name}")

    def decompose_task(self, task_description: str) -> List[Task]:
        """
        分解复杂任务为子任务
        """
        print(f"\n🎯 [{self.name}] 开始任务分解")
        print(f"   原始任务: {task_description}")

        # 模拟任务分解逻辑
        tasks = []

        # 任务1: 研究
        self.task_counter += 1
        task1 = Task(
            id=f"task_{self.task_counter:03d}",
            type="research",
            description="收集相关资料和数据",
            input_data={"topic": task_description},
            assigned_to="研究专家"
        )
        tasks.append(task1)

        # 任务2: 分析（依赖任务1）
        self.task_counter += 1
        task2 = Task(
            id=f"task_{self.task_counter:03d}",
            type="analysis",
            description="分析研究数据，提取洞察",
            dependencies=[task1.id],
            input_data={"depends_on": task1.id},
            assigned_to="分析专家"
        )
        tasks.append(task2)

        # 任务3: 写作（依赖任务2）
        self.task_counter += 1
        task3 = Task(
            id=f"task_{self.task_counter:03d}",
            type="writing",
            description="撰写报告内容",
            dependencies=[task2.id],
            input_data={
                "depends_on": task2.id,
                "research_id": task1.id,
                "title": task_description
            },
            assigned_to="写作专家"
        )
        tasks.append(task3)

        # 任务4: 审核（依赖任务3）
        self.task_counter += 1
        task4 = Task(
            id=f"task_{self.task_counter:03d}",
            type="review",
            description="审核报告质量",
            dependencies=[task3.id],
            input_data={"depends_on": task3.id},
            assigned_to="审核专家"
        )
        tasks.append(task4)

        print(f"\n   分解完成，共 {len(tasks)} 个子任务:")
        for t in tasks:
            deps = f" (依赖: {t.dependencies})" if t.dependencies else ""
            print(f"   - {t.id}: {t.type} → {t.assigned_to}{deps}")

        return tasks

    def execute_task(self, task: Task) -> Any:
        """执行单个任务"""
        agent = self.agents.get(task.assigned_to)
        if not agent:
            print(f"  ❌ 未找到Agent: {task.assigned_to}")
            return None

        task.status = "running"
        result = agent.execute(task)
        task.status = "completed"
        task.output_data = result

        # 存储结果
        self.shared_memory.store_task_result(task.id, result)

        return result

    def execute_tasks_parallel(self, tasks: List[Task]) -> Dict[str, Any]:
        """
        并行执行任务（处理依赖关系）
        """
        print(f"\n🚀 [{self.name}] 开始并行执行任务")

        results = {}
        completed_tasks = set()

        # 按依赖关系分批执行
        pending_tasks = tasks.copy()
        max_iterations = len(tasks) * 2  # 防止无限循环
        iteration = 0

        while pending_tasks and iteration < max_iterations:
            iteration += 1

            # 找出可以执行的任务（依赖已满足）
            ready_tasks = [
                t for t in pending_tasks
                if all(dep in completed_tasks for dep in t.dependencies)
            ]

            if not ready_tasks:
                if pending_tasks:
                    print("  ⏳ 等待依赖任务完成...")
                    time.sleep(0.5)
                    continue
                break

            print(f"\n  📦 本轮可执行任务: {[t.id for t in ready_tasks]}")

            # 并行执行就绪的任务
            with ThreadPoolExecutor(max_workers=4) as executor:
                futures = {
                    executor.submit(self.execute_task, task): task
                    for task in ready_tasks
                }

                for future in as_completed(futures):
                    task = futures[future]
                    try:
                        result = future.result()
                        results[task.id] = result
                        completed_tasks.add(task.id)
                        pending_tasks.remove(task)
                    except Exception as e:
                        print(f"  ❌ 任务 {task.id} 执行失败: {e}")
                        task.status = "failed"

        return results

    def integrate_results(self, results: Dict[str, Any], final_task_id: str) -> str:
        """
        整合各Agent的结果
        """
        print(f"\n🔧 [{self.name}] 开始结果整合")

        # 获取最终报告
        final_report = self.shared_memory.read(f"report_{final_task_id}")
        review = self.shared_memory.read(f"review_{final_task_id.replace('writing', 'review')}")

        if not final_report:
            return "报告生成失败"

        integration_result = f"""
{final_report}

---
## 质量审核报告

{review.get('feedback', '无审核反馈') if review else '未审核'}

**总体评分**: {review.get('score', 'N/A') if review else 'N/A'}/100

**审核状态**: {'✅ 通过' if review and review.get('approved') else '⚠️ 需要修改'}
"""

        print(f"  ✅ 结果整合完成")
        return integration_result

    def execute(self, task_description: str) -> str:
        """
        执行完整的多智能体协作流程
        """
        print("="*60)
        print("🤖 多智能体协作系统启动")
        print("="*60)

        start_time = time.time()

        # Step 1: 任务分解
        tasks = self.decompose_task(task_description)

        # Step 2: 并行执行任务
        results = self.execute_tasks_parallel(tasks)

        # Step 3: 结果整合
        final_task = tasks[-1]  # 最后一个是审核任务
        final_output = self.integrate_results(results, final_task.id.replace('review', 'writing'))

        elapsed_time = time.time() - start_time

        print("\n" + "="*60)
        print(f"✅ 任务完成！总耗时: {elapsed_time:.2f}秒")
        print("="*60)

        return final_output
```

## 几个实用技巧

- **角色设计**：Agent 的角色要清晰，避免职责重叠
- **依赖管理**：明确任务依赖关系，确保执行顺序正确
- **超时处理**：设置任务超时，防止某个 Agent 卡住整个流程
- **错误恢复**：某个 Agent 失败时，要有回退或重试机制
- **消息广播**：用消息机制替代轮询，提高通信效率
- **结果缓存**：已完成的结果缓存起来，避免重复计算
- **动态调整**：根据执行结果动态调整后续任务

# ❄️ 冷知识

1. **多智能体 vs 多线程**：虽然多智能体是用多线程实现的，但它们在抽象层次上完全不同。多线程是技术实现，多智能体是协作模式。就像你用微信建群聊(多智能体)，底层用的是网络传输(多线程)，但你能感受到的是群协作的便利。

2. **AutoGen 的多 Agent 架构**：微软的 AutoGen 框架是多智能体协作的典范。它内置了`UserProxyAgent`(用户代理)、`AssistantAgent`(助手 Agent)、`GroupChatManager`(群聊管理器)等角色，让多 Agent 协作变得非常简单。

3. **CrewAI 的理念**：CrewAI 这个名字就很有意思——把多 Agent 协作比作"船员团队"。每个 Agent 有自己的角色(Role)、目标(Goal)、背景故事(Backstory)，通过"任务(Task)"连接在一起完成共同使命。

4. **智能体间的共识问题**：多 Agent 系统面临一个经典问题——如何让多个 Agent 达成共识？这就像让你和朋友商量去哪吃饭，需要协调。研究者提出了"智能体谈判"、"投票机制"等解决方案。

5. **人类在循环中(Human-in-the-loop)**：实际应用中，多智能体系统往往不会完全自动运行，而是在关键节点让人类参与决策。比如审核 Agent 发现问题时，会暂停流程等待人类确认。
